# La canción del pirata

La canción del pirata es una aplicación web para la gestión de textos poéticos, que proporciona al usuario un sistema que permita escribir y maquetar poemas de manera simple, mediante un lenguaje de marcado específicamente diseñado para dar soporte a las necesidades de disposición más usuales, consultar los poemas de otros usuarios y poder crear colecciones con sus poemas favoritos.

## Despligue

Es necesario crear un archivo `.env`.

- ENVIRONMENT
- PROJECT_NAME
- STACK_NAME
- FRONTEND_HOST
- BACKEND_CORS_ORIGINS
- SECRET_KEY
- FIRST_SUPERUSER
- FIRST_SUPERUSER_PASSWORD
- FIRST_SUPERUSER_NAME
- SMTP_HOST
- SMTP_USER
- SMTP_PASSWORD
- EMAILS_FROM_EMAIL
- SMTP_TLS
- SMTP_SSL
- SMTP_PORT
- POSTGRES_SERVER
- POSTGRES_PORT
- POSTGRES_DB
- POSTGRES_TEST_DB
- POSTGRES_USER
- POSTGRES_PASSWORD
- IMAGES_DIR

El despliegue se realiza ejecutando `docker compose`.
